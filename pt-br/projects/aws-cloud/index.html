AWS Cloud

Área: nuvem
Data: 2018-2022

Este projeto envolveu a implementação de sites dinâmicos na AWS, gerenciamento de rede, DNS, snapshots e atualização de sistema em instâncias Ubuntu Server. Para isso, utilizei uma combinação de recursos da AWS, como Route 53 e Lightsail, juntamente com ferramentas remotas e de segurança. Durante a operação, também lidei com tarefas de gerenciamento de WordPress, criação de conteúdo e design e personalização de sites.

  • Ubuntu Server
  • Route 53
  • Lightsail
  • CloudWatch
  • Bitnami
  • Wordpress Multisite
