Homelab

Área: auto-hospedagem
Data: 2022-presente

Este projeto é um servidor doméstico customizado que envolve uma ampla gama de áreas, incluindo virtualização, conteinerização, segurança de rede, administração de sistema e infraestrutura como código. Ao integrar tecnologias como Ansible e Podman, tenho testado e implementado continuamente uma variedade de ferramentas e serviços, o que me permite criar um sistema otimizado para atender minhas demandas.

Sistema

  • QEMU/KVM
  • Ansible
  • Debian, RHEL
  • Docker, Podman

Rede

  • Nginx Reverse Proxy
  • Unbound DNS
  • Pihole
  • WireGuard
  • OpenSSH
  • iptables

Serviços

  • Home Assistant
  • Jellyfin
  • Hugo
  • Gotify

Gerenciamento

  • Portainer
  • Grafana
  • Prometheus
  • Node Exporter
  • cAdvisor
  • Watchtower
